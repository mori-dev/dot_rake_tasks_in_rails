# DotRakeTasksInRails

Add rake tasks at "~/.rake/*" to your Rails project.

## Installation

Add this line to your application's Gemfile:

```
gem 'dot_rake_tasks_in_rails'
```

And then execute:

```
$ bundle install
```

Or install it yourself as:

```
$ gem install dot_rake_tasks_in_rails
```
